```
        int b = 2;
        int a = 1;

            sum=a+b;
            a=b;
            b=sum;
            System.out.println(sum);
        }
```

```
private static int recursive(int n)
    {
        if (n == 1)
            return 1;
        else if ( n == 2)
            return 2;
        else
            return recursive(n - 1) + recursive(n - 2);
    }
```
